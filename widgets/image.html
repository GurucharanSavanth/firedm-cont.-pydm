image_value_changed(new_image)[source]¶

Callback invoked when the Image Channel value is changed. We try to do as little as possible in this method, because it gets called every time the image channel updates, which might be extremely often. Basically just store the data, and set a flag requesting that the image be redrawn.

Parameters:new_image (np.ndarray) – The new image data. This can be a flat 1D array, or a 2D array.
